## 🔥 Features

- **GENERATE SOUL** — one button generates a complete production blueprint
- **Chord Progression** — jazzy R&B and trapsoul voicings (maj9, m11, 13sus4 and beyond)
- **Mood Description** — cinematic, emotional scene-setting for your session
- **BPM Recommendation** — tempo matched to the vibe
- **Frequency Oasis Signal Chain** — curated combos using real FO VSTs with direct Gumroad links
- **Texture Chamber** — analog and lo-fi texture descriptions
- **▶ PLAY CHORD** — hear the chord progression via authentic Rhodes FM synthesis (Web Audio API)
  - FM carrier + modulator architecture
  - Velocity-sensitive bark attack (tine character)
  - Tremolo LFO at 5.1Hz
  - Spring reverb with warm EQ stack
- **Copy Session** — full blueprint to clipboard
- **Save Session** — persists locally across browser sessions
- **Cinematic UI** — gold luxury aesthetic, floating particles, animated waveform, custom cursor, glassmorphism

## 🛠 Tech Stack

- **HTML5 / CSS3 / Vanilla JavaScript** — zero frameworks, fully self-contained
- **Tailwind CSS** (CDN) — utility styling
- **Web Audio API** — Rhodes FM synthesis engine built from scratch
- **Canvas API** — ambient particle system + animated waveform
- **Google Fonts** — Cormorant Garamond, Bebas Neue, DM Mono, Rajdhani
- **localStorage** — session persistence

No build tools. No dependencies. One file. Open in any browser.
